About SwipeScapes

The Geospatial AI Copilot for Personalised Travel Intelligence

Founded by an IISc graduate and former McKinsey & Microsoft Geospatial Data Scientist with 8+ years of domain expertise and personal travel experience across 13+ countries, SwipeScapes eliminates Destination Paralysis and Itinerary Chaos. We automate the journey from travel inspiration to executable plans using proprietary geospatial technology and AI.

Core Travel Planning Solutions

Where-to-Go Index

For a given date, SwipeScapes evaluates visa rules, safety, seasonality, and public-transit mobility, and recommends where a user can go by matching these constraints with destination–user interest alignment. Destination personalities are inferred using Google Places POI data (place types, ratings, reviews, density) and mapped to user interests (e.g., culture, nightlife, wildlife, family-friendly)

Personalised Maps & Bucket-List Intelligence

Aggregates friend and community recommendations into live maps; converts saved social content into geocoded locations and layers feasibility, seasonality, and crowd-awareness insights on top.

Swipe-to-Plan Itinerary Optimizer

Users swipe to select places they’re interested in; SwipeScapes’ proprietary planning engine then generates day-by-day itineraries using Google Places data (ratings, reviews, opening hours, popularity signals) to build realistic, time-aware daily plans for the selected places.
